# AI Klarity 🚀  
### AI Powered Resume Analyzer & Interview Preparation Platform

AI Klarity is a full-stack AI powered web application designed to help students, job seekers, and developers improve their resume quality and prepare for interviews intelligently.

The platform allows users to upload resumes, analyze ATS compatibility score, extract technical skills automatically, detect probable job roles, and generate personalized mock interview preparation based on resume content.

The goal of this project is to provide a completely free AI-driven career preparation platform for users preparing for internships and job opportunities.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Dineshindiexpert/AI-KLARITY.git
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## 💡 Why I Built This Project

While applying for internships and job opportunities, I noticed that many resume analysis and interview preparation platforms were either paid, limited, or not personalized.

I wanted to build a completely free platform that helps students and developers analyze their resumes, improve ATS compatibility, and prepare for interviews based on their own resume content.

This project was built as a real-world solution for job seekers who need better preparation tools without paying expensive subscription fees.
